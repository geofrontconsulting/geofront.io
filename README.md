# geofront.io

## Pre-requirements

Please install `pipenv` to your python first.

## Install

```shell
git clone git@github.com:geofrontconsulting/geofront.io.git
cd geofront.io
pipenv install
```

## Launch locally

```shell
pipenv run serve
```

Open http://localhost:8000 in browser.

## Build

```shell
pipenv run build
```
